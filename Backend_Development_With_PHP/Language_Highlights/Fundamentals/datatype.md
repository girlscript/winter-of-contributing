### String
- A string is a sequence of characters.
- e.g. `Girlscript Winter Of Code`.
- We can put a string inside a double or single quote.

##### Code Snippet :

```php
<?php
$example = "Girlscript Winter Of Code";

echo $example;
echo "<br>";
echo "Hello Folks !"
?>
```
